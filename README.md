## 什么是 DU Meter？能监控什么？

DU Meter 是一款专为 Windows 设计的实时网络流量监控软件，由 Hagel Technologies, Ltd. 开发，已有超过二十年的历史。

它做的事情说白了就一件：盯着你电脑的网口，把所有进出的数据都记录下来，然后以你看得懂的方式展示出去。

具体能监控的内容包括：

- **实时网速**：当前上传速度 + 下载速度，精确到 KB/s 或 Mbps
- **累计流量**：今日、本周、本月用了多少
- **每个应用的流量**：哪个软件在偷跑流量，一眼看清
- **网络连接状态**：当前活跃的 TCP/UDP 连接，类似图形化的 netstat
- **流量费用估算**：按你的套餐价格，算出这个月大概花了多少钱

根据少数派对 DU Meter 的实测介绍，软件支持标准窗口、迷你窗口和任务栏三种监控显示模式，能同时监控互联网流量、局域网流量和本机应用流量，可满足从普通用户到服务器管理员的不同使用场景。

## DU Meter 核心功能详解

### 任务栏实时网速集成

这是很多用户选 DU Meter 的核心原因。右键点击 Windows 任务栏 → 勾选工具栏中的「DU Meter」，实时网速就会嵌入任务栏，不占桌面空间，和系统融为一体。

显示格式支持三种：纯文字、纯图形、文字+图形组合，看你喜欢哪种。

### 流量超限预警（两种模式）

DU Meter 提供短期警报和长周期警报两种模式。

**短期警报**：针对当前连接，可以对流量总量、上传量、下载量分别设置阈值，超过就弹出通知。

**长周期警报**：针对账单周期的累计流量。比如设置「本月入站+出站超过 50GB 时发邮件提醒我」，或者「当月流量费用超过 100 元时弹出提示」。如果你用的是计费套餐或者移动热点，这个功能非常实用。

自 DU Meter 5 开始，满足警报条件后软件甚至可以自动断网，无需人工干预。

### 多设备云端统计 Dashboard

在公司有台工作机、家里有台个人电脑，想在一个地方看所有设备的流量？DU Meter 提供免费的在线 Dashboard 服务（dumeter.net），把多台设备的流量数据汇总展示，每个设备的分时统计、月度账单都能在一个页面里看到。

### 详细流量报告导出

DU Meter 的报告支持按小时、日、周、月多个维度查看，并且可以导出为 RTF、XLS、PDF、HTML、TXT、XML、CSV 等多种格式——用 Excel 做进一步分析完全没问题。

### IP 地址过滤

可以将特定 IP 排除在统计之外，比如内网 NAS 的流量、公司局域网的数据同步。这样公网流量统计就不会被内网数据"污染"，对需要精确计算外网费用的用户来说很有用。

## DU Meter 常见问题解答

**Q：DU Meter 支持 macOS 吗？**  
目前 DU Meter 只支持 Windows，暂无 macOS 版本。需要 Mac 流量监控的用户可以在数码荔枝搜索对应平台软件。

**Q：电脑没有登录 Windows 账户时，DU Meter 还在监控吗？**  
是的。DU Meter 服务在 Windows 启动时就会运行，不需要登录账户也会持续监控。如果设置了邮件警报，提醒邮件同样会正常发送；但弹出通知需要用户登录后才能显示。

**Q：月中更改了流量套餐限额，之前的统计数据怎么算？**  
新的流量限额立即生效，已统计的流量数据会根据新设定自动更新，不会从零重新计算。

**Q：DU Meter 支持统计虚拟机的流量吗？**  
取决于虚拟机的网络模式。如果虚拟机使用桥接网络，DU Meter 无法统计（因为桥接发生在第二层，绕过了 DU Meter 运行的 IP 层）。可以将 DU Meter 直接安装在虚拟机内，结合 Dashboard 功能与主机数据合并查看。

**Q：购买后如何支付？支持哪些方式？**  
数码荔枝支持微信支付、支付宝、信用卡（Stripe）和 PayPal，付款后自动发货，可在「已购软件」页面查收激活码。
